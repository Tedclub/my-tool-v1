一、 策略核心參數作用與對應位置說明

此處解析程式碼中所有核心變數與參數，並指引您在 MetaTrader (MT5) 平台或程式碼中的具體位置：

參數/變數名稱 在 MQL5 中的程式碼位置 實際作用與代表意義 MT5 平台對應/尋找方式
SMAD (20) iMA(_Symbol,_Period,20,0,MODE_SMA,PRICE_CLOSE) **主移動平均線 (SMA)**。週期為 20，取收盤價。用作判斷短期趨勢的核心基準。 導航窗口 -> 技術指標 -> 趨勢指標 -> Moving Average。期間填入 20，平滑類型選 Simple。
SSMAD (20, Shift 20) iMA(_Symbol,_Period,20,20,MODE_SMA,PRICE_CLOSE) **位移移動平均線 (Shifted SMA)**。週期同樣是 20，但**向右平移 20 個 K 線單位**。與主 SMA 交叉時產生訊號。 同上，但在 Moving Average 的參數設定視窗中，將「位移 (Shift)」欄位填入 20。
SIXMAD (276) iMA(_Symbol,_Period,276,0,MODE_SMA,PRICE_CLOSE) **大週期過濾線 (XMAD)**。週期高達 276。用來當作長期趨勢的過濾器（程式碼中用於判斷當前價格是否在該線之上或之下）。 同 Moving Average 設定，將期間 (Period) 調整為 276。通常用於過濾掉逆勢的微小波動。
ATRD (14) iATR(_Symbol,_Period,14) **真實波幅均值指標 (ATR)**。週期 14。用來衡量市場當前的波動劇烈程度。 導航窗口 -> 技術指標 -> 震盪指標 -> Average True Range。期間設定為 14。
ATRV > 10 / < 80 if(ATRV>10) if(ATRV<80) **波動度過濾門檻**。限制只有在 ATR 值介於 10 到 80 點之間（非極端死寂也非極端暴動）時，才允許進場。 此為程式碼特有邏輯。在 MT5 執行 EA 時，需至「輸入參數 (Inputs)」分頁中尋找（若作者有導出為 input），否則需在原始碼中修改。
止損 (SL) / 止盈 (TP) (Ask-(ATRV*210)*_Point)
(Ask+(ATRV*800)*_Point)		 **動態風控點數**。
買入時：止損設在現價下方 $ATR \times 210$ 點；止盈設在現價上方 $ATR \times 800$ 點。		 這屬於動態點數計算。在 MT5 終端下單後，於「工具箱 -> 交易」分頁中的該筆訂單之 S/L 與 T/L 欄位可以看到最終計算出的價格。

二、 核心交易邏輯解析

買入訊號 (BUY) 成立條件

  • ATR 波動度必須適中：10 < ATRV < 80
  • 前一根 K 棒收盤價高於大週期過濾線：priceinfo[1].close > XMAD[0]
  • 前一根 MA 交叉向上：mymovingaveragearray[1] > myshiftedaveragearray[1]
  • 前兩根 MA 為交錯狀態：mymovingaveragearray[2] < myshiftedaveragearray[2]

賣出訊號 (SELL) 成立條件

  • ATR 波動度必須適中：10 < ATRV < 80
  • 前一根 K 棒收盤價低於大週期過濾線：priceinfo[1].close < XMAD[0]
  • 前一根 MA 交叉向下：mymovingaveragearray[1] < myshiftedaveragearray[1]
  • 前兩根 MA 為交錯狀態：mymovingaveragearray[2] > myshiftedaveragearray[2]

四、 JavaScript 核心邏輯實作對照 (網頁運作原理)

以下為網頁內建、完全對照您 MQL5 邏輯寫成的 JavaScript 代碼片段，展示如何不用 Python 即刻計算訊號：

// 網頁端模擬 MQL5 的 OnTick() 核心邏輯
function checkSignal(priceinfo, mymovingaveragearray, myshiftedaveragearray, XMAD, ATRV) {
    let signal = "";
    
    // 買入條件檢查
    if (ATRV > 10 && ATRV < 80) {
        if (priceinfo[1].close > XMAD[0]) {
            if (mymovingaveragearray[1] > myshiftedaveragearray[1]) {
                if (mymovingaveragearray[2] < myshiftedaveragearray[2]) {
                    signal = "buy";
                }
            }
        }
    }

    // 賣出條件檢查
    if (ATRV > 10 && ATRV < 80) {
        if (priceinfo[1].close < XMAD[0]) {
            if (mymovingaveragearray[1] < myshiftedaveragearray[1]) {
                if (mymovingaveragearray[2] > myshiftedaveragearray[2]) {
                    signal = "sell";
                }
            }
        }
    }
    return signal;
}
